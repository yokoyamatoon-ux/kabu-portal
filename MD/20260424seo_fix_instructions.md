### 1. コラム記事上部への広告表示テキスト追加

**問題**
Aboutページおよびコラム各記事の上部に、景品表示法が求める「広告を含む旨」の明示テキストが確認できない。アフィリエイトバナーは設置されているが、法令上はテキストによる明示が必要。

**対応内容**
全コラム記事（`col_001` 〜 `col_014`）およびAboutページの本文最上部に、以下のような告知バナーを追加する。

```html
<div class="ad-notice">

</div>
```

**確認基準**
- ページをブラウザで開いたとき、本文コンテンツより上に広告表示が目視できること
- CSSで視認性が確保されていること（文字サイズ・色が本文と判別できること）

### 3. JSON-LDの正確性検証

**問題**
JSON-LDはHTMLソースレベルでの確認が必要なため、外部からの検証が完了していない。実装ミスがあると構造化データがGoogleに認識されない。

**対応内容**

以下のツールで **最低3ページ** を検証すること。

- **Googleリッチリザルトテスト**：https://search.google.com/test/rich-results
- **Schema Markup Validator**：https://validator.schema.org/

**検証対象URL**

| ページ | 期待される構造化データ |
|---|---|
| `/column/col_013/` | `Article` + `FAQPage` |
| `/manga/1/` | `Article` |
| `/about/` | `Person` |

**確認基準**
- エラー・警告がゼロであること
- `Article`・`FAQPage`・`Person` の各タイプが正しく検出されること
- `author`・`datePublished`・`headline` フィールドが欠損していないこと

### 4. 固定ページのdescriptionが個別設定されているか確認

**問題**
About・Manga・Column等の一覧ページのtitleが全ページ共通になっており、`<meta name="description">` も共通の可能性がある。descriptionが重複するとSEO上のペナルティとなる場合がある。

**対応内容**

ブラウザの「ページのソースを表示」または開発者ツールで、各固定ページの `<head>` 内を確認する。

**確認対象ページとdescription例**

| ページ | 推奨description（100〜120文字） |
|---|---|
| `/` | マンガとクイズでお金の基本が学べる入門サイト。NISA・株・投資信託をカブ先生がわかりやすく解説。 |
| `/about/` | カブ先生のお金の学校とは何か。サイトのコンセプト・学べること・登場キャラクターを紹介します。 |
| `/manga/` | 投資の基本をマンガで楽しく学ぼう。株・NISA・配当金など、難しい言葉ゼロで解説する全5話。 |
| `/column/` | 経済ニュースや投資の疑問をカブ先生が解説するコラム一覧。初心者にもわかりやすい記事を毎週更新。 |
| `/quiz/` | 投資の知識をクイズ形式で楽しく確認。NISA・株・経済用語など全問解説付き。 |
| `/qa/` | 投資・お金に関する疑問をカブ先生に質問できるコーナー。初歩的な質問も大歓迎。 |
| `/money_secret/` | 知らないと損するお金の裏事情を解説。手数料の罠・詐欺的投資商品など、身を守る知識を提供。 |
| `/privacy/` | カブ先生のお金の学校のプライバシーポリシー。個人情報の取り扱いについて説明します。 |
| `/disclaimer/` | 当サイトの免責事項。投資判断は自己責任となります。詳細をご確認ください。 |
| `/contact/` | カブ先生のお金の学校へのお問い合わせフォーム。ご意見・ご要望をお気軽にどうぞ。 |

**確認基準**
- 全ページのdescriptionが固有の内容になっていること
- 80〜130文字の範囲に収まっていること
- キーワードが自然に含まれていること
