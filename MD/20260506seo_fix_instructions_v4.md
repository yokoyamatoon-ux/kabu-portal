### 【手順1】本番環境へのデプロイと動作確認（最優先）

**目的：** ローカル（localhost:3000）で検証済みの修正内容を本番サーバーに反映させる

**確認項目（デプロイ後に実施）：**

```
確認URL一覧：
https://okane-no-manabi.jp/
https://okane-no-manabi.jp/column/
https://okane-no-manabi.jp/column/col_017/
https://okane-no-manabi.jp/manga/
https://okane-no-manabi.jp/money_secret/
https://okane-no-manabi.jp/maneta_diary/
```

| 確認項目 | 確認方法 |
|---------|---------|
| H1タグの出力 | ブラウザのデベロッパーツール（F12）→ Elements でH1を確認 |
| パンくずの表示 | コラム詳細ページを開き「ホーム / コラム一覧 / 記事名」が表示されることを確認 |
| JSON-LD出力 | デベロッパーツール → Elements で `<script type="application/ld+json">` を確認 |
| robots.txt | `https://okane-no-manabi.jp/robots.txt` をブラウザで開いて確認 |
| sitemap.xml | `https://okane-no-manabi.jp/sitemap.xml` をブラウザで開いて確認 |

**JSON-LD動作検証：**  
[Googleリッチリザルトテスト](https://search.google.com/test/rich-results) に本番URLを入力してエラーがないことを確認する。

### 【手順3】見出し階層（Hn）の確認

**目的：** H1設置と同時に、H2・H3の階層が正しく構成されているかを確認する

H1は設置済みですが、その下の見出し構造が正しいか本番で確認してください。

**確認ポイント：**

```html
<!-- ✅ 正しい構造 -->
<h1>お金の学び｜初心者向けお金・投資の完全ガイド</h1>
  <h2>投資の基本</h2>
    <h3>株式投資とは</h3>
  <h2>節約の考え方</h2>

<!-- ❌ NGパターン（H1の次にH3が来ている） -->
<h1>お金の学び</h1>
<h3>投資の基本</h3>
```

**確認ツール：** ブラウザ拡張「HeadingsMap」を使うと全ページの見出し構造を一覧表示できます。

## 🟠 引き続き対応が必要な項目

### 【手順4】コンテンツの拡充（E-E-A-T強化）

**目的：** コンテンツスコア49点（業界平均71.7）を改善する

JSON-LDは設置済みのため、次はコンテンツ自体の質を上げる段階です。

**優先的に対応するコンテンツ：**

| コンテンツ | 対応内容 |
|-----------|---------|
| 各コラム記事 | 2,000文字以上に拡充・著者プロフィールを記事内に明記 |
| マンガ各話 | エピソードの説明文を拡充・関連コラムへの内部リンクを追加 |
| ウラ金（money_secret） | 参考文献（金融庁・公式サイト）へのリンクを追加 |
| マネタ日記 | 各回の最終更新日を表示 |

**Answer First形式（AI検索対策）への書き換え：**

```
❌ 従来形式：
「投資とは何でしょうか。歴史的には...（長い前置き）...つまり資産を増やすことです」

✅ Answer First形式（冒頭30〜50文字で結論）：
「投資とは、将来の利益を目的に資金を運用することです。
以下で初心者向けの始め方を解説します。」
```

---

### 【手順5】各記事へのFAQセクション追加

**目的：** FAQPage スキーマはすでに設置済みのため、HTML上にもFAQコンテンツを可視化する

各記事の末尾に「よくある質問」を3〜5問追加する：

```html
<section class="faq-section">
  <h2>よくある質問</h2>
  <dl>
    <dt>Q. 投資初心者は何から始めるべきですか？</dt>
    <dd>A. まずは少額でできる積立NISAや投資信託から始めるのがおすすめです。</dd>

    <dt>Q. いくらから投資を始められますか？</dt>
    <dd>A. 証券会社によっては100円から積立投資が可能です。</dd>
  </dl>
</section>
```

> ⚠️ FAQPage JSON-LDの `mainEntity` の内容と、HTML上のFAQの内容を一致させること。

---

### 【手順6】内部リンクのさらなる強化

**目的：** 内部リンクスコアは73点（業界平均83.4）。パンくずは設置済みのため、記事間リンクを追加する。

**修正アクション：**

- 各コラム記事の本文中から、関連する他のコラム・マンガ・日記ページへのリンクを2〜5本追加する
- 「関連記事」セクションを各記事末尾に設置する
- トップページに「新着記事」「おすすめ記事」セクションを追加する

**関連コンテンツのリンク設置例：**

```html
<!-- コラム記事末尾に追加 -->
<section class="related-articles">
  <h2>関連記事</h2>
  <ul>
    <li><a href="/column/col_012/">初心者向け！投資信託の始め方</a></li>
    <li><a href="/manga/ep_006/">マンガで学ぶNISAの仕組み</a></li>
  </ul>
</section>
```
