# 更新マスターチェックリスト (最新版)
**最終更新日：** 2026年4月30日
**目的：** コンテンツ更新時のミスを防ぎ、SEO・AIO効果を最大化する。

---

## 1. コンテンツ追加・修正時 (共通)
- [ ] **データの保存先：** `data/` フォルダのJSONを編集したか？（`web-next/`内を直接編集してもデプロイ時に上書きされるので注意）
- [ ] **画像の配置：** `image/` フォルダ内の適切なサブディレクトリに画像を置いたか？
- [ ] **ステマ規制対応：** 記事冒頭に「※本ページには広告・アフィリエイトリンクが含まれます。」があるか？
- [ ] **キャラクター口調：** カブ先生（〜じゃ、〜のう）、マネ太（〜っす）のキャラ設定は守られているか？

## 2. コラム (columns.json) 更新時
- [ ] **AIO対策 (定義段落)：** 冒頭に「[キーワード]とは、〜です。」という明確な定義文を入れたか？
- [ ] **FAQセクション：** 記事末尾（カブ先生の一言の前）に3問のFAQがあるか？
- [ ] **内部リンク：** 関連する他のコラムやマンガへのリンク（📚 あわせて読みたい）があるか？
- [ ] **出典明記：** 公的機関（金融庁、日銀等）の出典元リンクを末尾に記載したか？

## 3. マンガ (manga.json) 更新時
- [ ] **解説文テキスト：** 画像の下に1,500〜2,000字程度の対話形式解説テキストを追加したか？
- [ ] **エモート設定：** `commentary` 配列の `emote` ("normal", "surprise") は適切か？
- [ ] **登場キャラ制限：** マンガセクションに「ウラ金さん」を登場させていないか？（カブ先生、マネ太、ミライのみ）

## 4. デプロイ作業
- [ ] **JSONバリデーション：** `python -m json.tool data/xxx.json` で文法エラーがないか確認したか？
- [ ] **デプロイ実行：** `python deploy_kabu.py` をプロジェクトルートで実行したか？
- [ ] **表示確認：** 本番サイトで **Ctrl + F5** (強制リロード) を行い、変更が反映されているか？
- [ ] **リンク確認：** 追加したページのリンクが404になっていないか？

## 5. SNS (X) 投稿時
- [ ] **リンク制限：** 親ツイートにURLを含めていないか？（URLはリプライ欄に貼る）
- [ ] **キャラ設定：** カブ先生の「フォッフォッフォ！」「〜ぞい」の口調になっているか？
- [ ] **画像添付：** インパクトのある図解やマンガ画像を1枚以上添付したか？
