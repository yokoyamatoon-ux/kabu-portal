# マンガページ 解説文 作成指示書

**作業ツール：** Antigravity（Google）  
**作成日：** 2026年4月22日

## ▼ 作業の目的

各マンガページ（第1〜5話）に「本文テキスト解説」を追加する。  
現状は画像のみで本文テキストがゼロのため、SEO・AIO対策として必須の作業。

---

## ▼ 文体・テイストのルール（必ず守ること）

参考：`okane-no-manabi.jp/column/col_013/`（S&P500 vs オルカン記事）

| 項目 | ルール |
|------|--------|
| 語り手 | カブ先生（老賢者口調）。語尾は「〜じゃ」「〜のじゃ」「〜ぞ」 |
| 聞き手 | マネ太（元気な若者口調）。語尾は「〜っす」「〜ですか！？」 |
| 難易度 | 中学生でもわかるレベル。専門用語には必ずカッコで説明を入れる |
| 文体 | 対話形式（カブ先生とマネ太の掛け合い）＋まとめブロック |
| 絵文字 | 使用OK。見出しや箇条書きのアイコンとして活用 |
| 文量 | 各話 1,500〜2,000字程度 |

---

## ▼ 各話の構成テンプレート

各話は以下の4ブロックで構成すること。

```
【ブロック1】導入の対話（カブ先生とマネ太のやりとり・約300字）
【ブロック2】核心の解説（この話のメインテーマを説明・約600字）
【ブロック3】まとめ（箇条書きで3〜5項目・約300字）
【ブロック4】FAQセクション（Q&A形式で2〜3問・約400字）
```
