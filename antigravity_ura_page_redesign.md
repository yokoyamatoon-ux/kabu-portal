# Antigravity 修正プロンプト
## 「裏事情」ページのレイアウトをマンガページと同じ構造に変更する

---

## 現状と目標の差分

### 現在の「裏事情」ページの構造
```
① ヘッダー画像（ウラ金さんのお金の裏事情ファイル）
② ウェルカムテキストBOX（赤枠）
③ 黒背景の本編エリア
   - テーマタイトル「今回のテーマ：インサイダー取引」
   - テーマ画像（夜の路地裏）
   - キャラクター吹き出し会話×3
④ +ホームにもどる ボタン
```

### 目標：マンガページと同じ構造
```
① ページタイトル（左揃え・大きめ）
② エピソードカード一覧（2カラムグリッド）
   各カード：
   - サムネイル画像（上部・横長）
   - エピソード番号ラベル（赤・小さめ）
   - エピソードタイトル（太字）
   - サマリーテキスト（グレー・小さめ）
   - 「第○話を読む →」ボタン（カード下・全幅）
```

---

## 変更対象ファイル

裏事情ページを担当しているファイルを特定してすべて変更する。
マンガページのコードをベースに、裏事情用のデータに差し替えること。

---

## ① ページタイトル

マンガページのタイトルは `📖 マンガで学ぶ` 。
裏事情ページのタイトルは以下に変更する：

```
🕵️ お金の裏事情ファイル
```

---

## ② エピソードデータ

マンガページの `manga_episodes` リストと同じ構造で、以下のデータを定義する。
変数名は `ura_episodes` とする。

```python
ura_episodes = [
    {
        "ep": 1,
        "label": "第1話",
        "title": "インサイダー取引",
        "summary": "秘密の情報で株を買うのは立派な犯罪じゃ！",
        "image_path": "（既存のインサイダー取引テーマ画像のパスをそのまま使う）",
        "btn_text": "第1話を読む →",
        "link_target": "ura_ep1",   # 既存の詳細ページがあればそのキーを使う
    },
    # 以降のエピソードが既存コードにあれば同じ形式で追加する
    # エピソードがまだ1話しかない場合は1話だけでOK
]
```

> **注意：** 既存コードのエピソードデータ・画像パス・ページ遷移キーを
> そのまま移植すること。新しく作らなくてよい。

---

## ③ カードのレンダリング

マンガページの `render_manga_card()` 関数（または同等の描画処理）を
そのまま流用し、`ura_episodes` を渡して描画する。

カードのデザイン仕様はマンガページと完全に同一にすること：

| 要素 | 仕様 |
|------|------|
| グリッド | 2カラム（`st.columns(2)`） |
| サムネイル | カード上部・横長・角丸 |
| エピソード番号 | 赤文字・小さめ（`font-size: 0.75rem`） |
| タイトル | 太字・`font-size: 1rem` |
| サマリー | グレー文字・`font-size: 0.85rem` |
| ボタン | カード下部・全幅・`第○話を読む →` |
| 背景 | 白カード・角丸・薄いシャドウ |
| ページ背景 | クリーム色（マンガページと同じ） |

---

## ④ 削除する要素

以下は新レイアウトでは**不要**なので削除する：

- ウェルカムテキストBOX（赤枠の「「お金の裏事情」へようこそ…」）
- 黒背景の本編エリア全体
- 吹き出し会話UI（`.bubble-left` / `.bubble-right`）
- 「+ホームにもどる」ボタン
- ヘッダー画像（ウラ金さんのお金の裏事情ファイル）

---

## ⑤ 詳細ページへの遷移

「第○話を読む →」ボタンを押したときの挙動は**既存の遷移ロジックをそのまま維持**する。
マンガページのボタンと同じ方式（`st.session_state` のページキー切り替えなど）で実装する。

---

## 変更してはいけない部分

- マンガページのコードは一切変更しない
- ナビゲーション（ヘッダーのタブ）は変更しない
- 他のページ（ホーム・クイズ・探す・質問箱・投資日記）は変更しない
- 既存の裏事情詳細ページ（インサイダー取引の本文・吹き出し会話）は残す
  （一覧ページからリンクで飛べるようにする）

---

## 確認ポイント

1. 裏事情ページを開いたとき、マンガページと同じカードグリッドが表示される
2. カードのサムネイルに既存の画像が表示される
3. 「第1話を読む →」ボタンで既存の詳細ページに遷移できる
4. マンガページに変化がない
5. ページ背景がクリーム色になっている（黒背景が消えている）
