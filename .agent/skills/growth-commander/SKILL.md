## 📋 新規記事公開チェックリスト

記事を公開するたびに、以下の手順を必ず実行すること。

### Step 1: GEO（AI検索最適化）チェック
記事の冒頭に以下の「サマリーボックス」が存在するか確認する：
```
> **📋 この記事でわかること**
> - **[ポイント1]：** [説明]
> - **[ポイント2]：** [説明]
> - **[ポイント3]：** [説明]
> 🔑 **キーワード：** [キーワード / キーワード / キーワード]
```
→ 存在しない場合は追加してから `upload_article.py` を実行する。

### Step 2: アイキャッチ作成
- `articles/XX_topic/eyecatch/eyecatchXX.png` に画像が存在するか確認する。
- 存在しない（空ファイル）場合、FreePICでアイキャッチを生成して置き換える。

### Step 3: SNS投稿案の生成
```bash
# 記事公開後、即座に実行する（「SNSよろ！」と言えば自動実行）
python generate_sns.py articles/XX_topic/article.md
```
生成された `sns_drafts.txt` の**パターンA（エージェントAの悲哀スタイル）を優先的に採用**する。
トーンが合わない場合のみパターンB/Cを参照する。

### Step 4: 内部リンク確認
`upload_article.py` のアップロード時に自動的に内部リンクが付与される。
新記事と関連性の高い既存記事がある場合、その記事も `upload_article.py` で再アップロードして相互リンクを強化する。
