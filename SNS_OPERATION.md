# X（Twitter）運用マニュアル（カブ先生）

このドキュメントは、AIエージェントやプロジェクト担当者が「カブ先生」のSNSアカウントを継続的・安定的に運用するためのガイドラインです。セッションが切り替わっても、このファイルを最初に読み込むことで正しく運用を再開できます。

## 1. 運用戦略の基本
詳細は [sns_strategy_kabuteacher.md](file:///D:/Antigravity/X/sns_strategy_kabuteacher.md) を参照してください。

- **コンセプト**: 難しい投資をマンガと図解で日本一ポップに教える先生。
- **ペルソナ**: 親しみやすいおじいちゃん先生風。
  - **笑い方**: 「フォッフォッフォ！」
  - **語尾**: 「～ぞい」「～じゃ」「～のう」
- **投稿タイミング**: 
  - **朝（6:30〜7:30）**: マインド系、挨拶。
  - **夜（18:30〜20:30）**: 有益図解、コラム解説。

## 2. データの管理と参照
- **投稿履歴**: [data/sns_post_history.json](file:///d:/Antigravity/Kabu/data/sns_post_history.json)
  - 新しいAIがプロジェクトに入った際は、必ずこの履歴を確認し、**同じ内容を短期間に投稿しない**ようにしてください。
- **認証情報**: 全てのスクリプトはシステム環境変数（`KABU_X_...`）からキーを取得します。プロジェクト内に `.env` を置かないルールを徹底してください。

## 3. 主要スクリプトの役割
全てのスクリプトはルートディレクトリで実行することを想定し、相対パスで構成されています。

| ファイル名 | 役割 | 備考 |
| :--- | :--- | :--- |
| `post_to_x_kabu.py` | 共通投稿エンジン | 履歴への自動記録機能付き。 |
| `modules/sns_config.py` | 共通設定・ロジック | パス解決、文字数計算、環境変数取得など。 |
| `post_x_monday_mindset.py` | 月曜・夜の投稿用 | マインド系メッセージ + `Character.png` |
| `post_x_column_20260414.py` | 火曜・夜の投稿用 | コラム解説。スレッド（2枚目にリンク）構成。 |

## 4. AIエージェントへの指示（引継ぎ事項）
1. 投稿前に必ず `modules/sns_config.py` の `count_twitter_points` を使い、280ポイント以内であることを確認してください。
2. インプレッション低下を避けるため、**メインツイートにURLを含めず、必ずリプライ欄にぶら下げる** 構成を維持してください。
3. 投稿後は `data/sns_post_history.json` を読み込み、正しく記録されたか確認してください。
