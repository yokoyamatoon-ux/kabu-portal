# Antigravity 修正プロンプト
## ホームページ導入セクションを「超初心者向け」に刷新する

---

## ターゲットの再定義

**現在のターゲット想定（変更前）**
投資に多少興味がある人・株を調べたことがある人

**新しいターゲット（変更後）**
- 「投資ってなんとなく気になるけど怖い」
- 「NISAってよく聞くけど何なの？」
- 「お金のことよくわからないし、自分には関係ない気がする」

→ **「投資」という言葉を見た瞬間に閉じようとする人を引き止める**のが目的

---

## 追加・変更するセクション

ホームページの既存コンテンツの**上に**、以下の3つのセクションを追加する。
既存のコンテンツ（マンガ一覧・クイズ・探すへのリンクなど）はそのまま下に残す。

---

## セクション① 「あなたは大丈夫？」チェック（共感フック）

画面上部、ヒーローエリアのすぐ下に配置。

```
デザイン：カード形式・背景はクリーム〜薄オレンジのグラデーション

タイトル：「こんなこと、思ったことない？」

チェック項目（チェックマーク付きリスト）：
  ☑ 貯金はしてるけど、このままでいいのか不安…
  ☑ NISAってよく聞くけど、何をすればいいかわからない
  ☑ 投資って難しそうで、自分には無理そう
  ☑ お金のことを誰かにわかりやすく教えてほしい

締めのひとこと（カブ先生の吹き出し形式）：
  「1つでも当てはまったら、このサイトはあなたのためにあるじゃ！」
```

---

## セクション② 「なぜ今、投資が必要なの？」（危機感→納得）

セクション①の下に配置。

```
タイトル：「なぜ、銀行に預けるだけじゃダメなの？」

3つのカードを横並び（モバイルは縦積み）で表示：

カード1：💴 銀行の金利
  見出し：「銀行に100万円預けると…」
  本文：「1年後に増えるのは100円ほど。
        物価が上がる分、実質的にはお金が減っています。」

カード2：📈 インフレの現実
  見出し：「物価は毎年上がっている」
  本文：「2年で10〜20%以上値上がりした商品も多数。
        同じお金で買えるものが少なくなっている。」

カード3：🌱 投資との違い
  見出し：「お金を"働かせる"とどうなる？」
  本文：「年3〜5%で運用できれば、20年後には
        100万円が180万〜265万円になる計算に。」

注意書き（小さめ）：
※投資にはリスクがあります。上記はあくまで参考値です。
```

---

## セクション③ 「このサイトで何ができるの？」（サイト紹介）

セクション②の下に配置。

```
タイトル：「カブ先生の学校でできること」

4つの機能カードをグリッド表示（2×2、モバイルは1列）：

カード1：📖 マンガで学ぶ
  「むずかしい言葉ゼロ。
   カブ先生とキャラクターたちのマンガで
   株・NISA・配当金がサクッとわかる。」
  → 「マンガを読む」ボタン（マンガページへ遷移）

カード2：❓ クイズで試す
  「読んだら試してみよう。
   全問正解できたらあなたも立派な
   "投資初心者卒業生"じゃ！」
  → 「クイズをやってみる」ボタン（クイズページへ遷移）

カード3：🔍 銘柄を探す
  「実際にどんな会社があるの？
   テーマや条件から気になる銘柄を
   探して仮想投資体験もできる。」
  → 「銘柄を探す」ボタン（探すページへ遷移）

カード4：🕵️ 裏事情を知る
  「投資の"やってはいけない"も学べる。
   詐欺・インサイダー取引…
   知らないと損するダークサイドも解説。」
  → 「裏事情を見る」ボタン（裏事情ページへ遷移）

各ボタンは既存の st.session_state ページ遷移と同じ方式で実装する。
```

---

## デザイン仕様（既存サイトと統一）

| 要素 | 仕様 |
|------|------|
| ページ背景 | クリーム色（既存と同じ） |
| セクション背景 | 白カード・角丸16px・薄シャドウ |
| タイトルフォント | M PLUS Rounded 1c・太字（既存と同じ） |
| アクセントカラー | 既存の赤系プライマリカラーを使う |
| モバイル対応 | `st.columns` で2カラム → モバイルは自動1カラムに |

---

## 変更してはいけない部分

- 既存のヒーローエリア（カブ先生のキャラクター・キャッチコピー）は変更しない
- 既存のマンガ一覧・クイズ・その他のコンテンツはそのまま残す
- ナビゲーションは変更しない
- マンガ・クイズ・探す・裏事情の各ページ本体は変更しない

---

## 確認ポイント

1. ホームを開いたとき、ヒーローの下に3つの新セクションが順番に表示される
2. セクション③の各ボタンで正しいページに遷移できる
3. 既存コンテンツが3セクションの下にそのまま表示されている
4. スマホ幅でレイアウトが崩れない
5. 数値に関する注意書きが表示されている
