## 記事の追加方法

`modules/column_unit.py` の `COLUMNS` リストに以下の形式で辞書を追加するだけ。
画像は `image/column/col_XXX.jpg` に配置する（なくてもOK、プレースホルダー表示）。

```python
{
    "id": "col_XXX",              # 連番（col_001, col_002...）
    "date": "2026.03.23",         # 公開日
    "category": "カテゴリ名",      # 下記カテゴリ一覧から選ぶ
    "category_color": "#FF6B6B",  # 下記カテゴリ一覧から対応する色
    "title": "記事タイトル",
    "lead": "リード文（カード表示・3行以内）",
    "image_path": "image/column/col_XXX.jpg",  # サムネイル画像パス
    "body": """
記事本文（Markdown形式で書く）
    """,
    "tags": ["タグ1", "タグ2"],
    "reading_time": 3,            # 想定読了時間（分）
},
```
